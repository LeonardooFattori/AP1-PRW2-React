# 🖥️ Total Hardware — Loja de Informática

- Projeto desenvolvido pelos alunos do 4º período de ADS Leonardo Fattori e Felipe Candelora em React sendo a 1ª Avaliação Prática da disciplina PRW2. O objetivo foi criar um site de e-commerce para uma loja de informática, aplicando os conceitos estudados nas últimas três aulas.

## 📚 Conceitos Aplicados
- Criação e composição de componentes funcionais
- Uso de props para passagem de dados entre componentes
- Gerenciamento de estado com useState
- Renderização de listas com .map()
- Renderização condicional de conteúdo
- Filtragem de produtos por categoria e marca
